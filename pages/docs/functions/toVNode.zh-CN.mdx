# `toVNode`

**语法：** `toVNode(element)`\
**示例：** `toVNode(document.querySelector('#app'))`

`toVNode` 函数将 HTMLElement 或 Text 转换为 VNode。 这通常用于从 SSR 中重新水化 HTML。

```js
import { m, toVNode, patch } from 'million';

const el = document.querySelector('#app');
const vnode = m('div', { id: 'app' }, ['Hello World'], Flags.ONLY_TEXT_CHILDREN);
const oldVNode = toVNode(el);

patch(el, vnode, oldVNode);
```

```html
<div id="app">Hello World</div>
```
