＃ 安装

有两种方法可以将 Million 包含到您的项目中：

- 将其作为模块导入
- 从 `<script>` 标签中包含它

Million 默认情况下不需要构建工具，但强烈建议您使用 NPM 安装，尤其是在构建 UI 库时。

## 作为一个模块

如果您使用 NPM 安装 Million，请确保使用 [Vite](https://vitejs.dev) 或 [Rollup](https://rollupjs.org/) 等捆绑程序。

运行以下命令进行安装。

```sh
npm install million
```

现在，将 Million 导入到您的源文件中，如下所示：

```js
import {
  m,
  patch,
  createElement,
  /* or any other exports you want to access */
} from 'million';
```

## 来自脚本标签

这是迄今为止最简单的开始使用 Million 的方法。在 HTML 页面的头部包含以下 `<script>` 标记。然后，使用类似 UNPKG 的 CDN 并通过 ES 模块导入百万。

```html
<script type="module">
  import {
    m,
    patch,
    createElement,
    /* or any other exports you want to access */
  } from 'https://unpkg.com/million?module';

  // Your code here
</script>
```

而已！百万现在可以在您的页面内使用。
